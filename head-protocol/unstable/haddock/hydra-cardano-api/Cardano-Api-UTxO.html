Description

NOTE (1): This module is meant to be imported qualified as UTxO.

NOTE (2): This module is name-spaces slightly different from the rest because it is meant to be used as a replacement of the UTxO type of the cardano-api which is not convenient enough to work with. Having it as UTxO causes cyclic imports with other modules also relying on this newtype. So instead, we do 'as if' it was part of the cardano-api in the first place.
